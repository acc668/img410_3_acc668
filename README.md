## Usage
```
make
./raycast width height input.scene output.ppm
```

- `width` and `height` are the dimensions in pixels of the output image
- `input.scene` is the scene file to render
- `output.ppm` is the output image in P3 PPM format

Example:
```
./raycast 400 300 basic.scene output.ppm
```
